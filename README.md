# 프로젝트 9조 - 학습 도우미 웹사이트 📚

> 중고등학생을 위한 자기주도학습 지원 플랫폼 MVP

## 📋 프로젝트 개요

**프로젝트 9조**는 중고등학생들의 자기주도학습을 돕기 위한 웹 기반 학습 도우미 플랫폼입니다. 학생들이 체계적으로 공부 계획을 세우고, 집중력을 높이며, 입시 및 학습 관련 조언을 받을 수 있는 핵심 기능을 제공합니다.

### 🎯 타겟 사용자
- 중학생, 고등학생
- 자기주도학습이 어려운 학생
- 입시 정보와 공부 방법이 필요한 학생

## ✨ 주요 기능

### 1. 🏠 메인 홈
- 직관적인 3개 카테고리 네비게이션
- 오늘의 공부 시간 실시간 표시
- 오늘의 할 일 진행 상황 확인

### 2. 💬 컨설팅 받기
- **규칙 기반 AI 챗봇**: 입시 및 학습 관련 맞춤형 조언 제공
- **3가지 카테고리**:
  - 대입 컨설팅: 내신, 수능, 학종, 정시 등
  - 고입 컨설팅: 특목고, 자사고, 일반고 등
  - 학습 방법: 암기법, 노트정리, 집중력, 시험 준비 등
- 자주 묻는 질문 빠른 답변
- 대화 기록 자동 저장

### 3. ⏰ 공부 시작하기
- **뽀모도로 타이머**: 25분 집중 + 5분 휴식 자동 전환
- **커스텀 타이머**: 원하는 시간대로 자유롭게 설정
- **백색소음**: 빗소리, 카페 소음, 도서관 소음 선택 가능
- **실시간 통계**: 오늘의 공부 시간 및 연속 학습일 표시
- **동기부여**: 랜덤 명언 표시
- 브라우저 알림 지원

### 4. 📅 공부 계획 세우기
- **일간 플래너**: 날짜별 할 일 관리
- **체크리스트**: 완료 항목 체크 및 진행률 시각화
- **할 일 관리**: 추가, 삭제, 순서 변경, 완료된 항목 일괄 삭제
- **진행률 표시**: 실시간 프로그레스 바 및 통계
- 효과적인 계획 세우기 팁 제공

## 💾 데이터 구조

### RESTful Table API 스키마

#### 1. study_sessions (공부 세션)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| date | text | 날짜 (YYYY-MM-DD) |
| duration | number | 공부 시간 (분) |
| type | text | 타이머 타입 (pomodoro/custom) |

#### 2. daily_tasks (일간 할 일)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| date | text | 날짜 (YYYY-MM-DD) |
| task | text | 할 일 내용 |
| completed | bool | 완료 여부 |
| order | number | 정렬 순서 |

#### 3. chat_history (컨설팅 기록)
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| category | text | 카테고리 (대입/고입/공부법) |
| message | text | 사용자 메시지 |
| response | text | 챗봇 응답 |
| timestamp | datetime | 대화 시간 |

## 📊 API 엔드포인트

### RESTful Table API 사용 예시

```javascript
// 공부 세션 저장
POST /tables/study_sessions
Body: { id, date, duration, type }

// 할 일 목록 조회
GET /tables/daily_tasks?limit=100&sort=order

// 할 일 완료 처리
PATCH /tables/daily_tasks/{id}
Body: { completed: true }

// 할 일 삭제
DELETE /tables/daily_tasks/{id}
```

## 🐛 알려진 제한사항

- **챗봇**: 규칙 기반이므로 복잡한 질문에는 제한적인 답변 제공
- **백색소음**: Web Audio API 기반 합성음으로 실제 소리와 차이 있음
- **인증 시스템**: 현재 사용자 계정 기능 없음 (모든 데이터 공유)
- **알림**: 브라우저 권한 허용 필요

## 💡 사용 팁

1. **컨설팅 챗봇**: 구체적인 키워드(내신, 수능, 암기 등)를 사용하면 더 정확한 답변을 받을 수 있습니다.
2. **타이머**: 뽀모도로 모드는 25분 집중 후 자동으로 5분 휴식으로 전환됩니다.
3. **백색소음**: 볼륨은 하단 슬라이더로 조절 가능하며, 타이머와 독립적으로 작동합니다.
4. **플래너**: 날짜 네비게이션으로 과거/미래 날짜의 할 일도 관리할 수 있습니다.
5. **진행률**: 할 일을 체크할 때마다 실시간으로 진행률이 업데이트됩니다.
